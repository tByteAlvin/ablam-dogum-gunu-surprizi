# Ablamın Doğum Günü Sürprizi

Elif için hazırlanmış, telefon ve bilgisayarda çalışan etkileşimli doğum günü sitesi.

## İçerik değiştirme

- Yazılar ve dosya yolları: `src/config.js`
- Fotoğraflar: `assets/images`
- Çiçekler: `assets/flowers`
- Müzikler: `assets/music`
- Ses efektleri: `assets/sounds`

Dosya adlarını `src/config.js` içindeki adlarla aynı tutmak yeterli.
